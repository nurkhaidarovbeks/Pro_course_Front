## React Native vs React Web

React web uses DOM elements such as div, span, and p.

React Native uses native components such as:

- View
- Text
- Image
- TextInput
- TouchableOpacity
- ScrollView

React web uses CSS classes. React Native uses the StyleSheet API.

React web uses onClick. React Native uses onPress.
