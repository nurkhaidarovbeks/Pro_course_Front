## Description

This project is a Todo application tested with Jest and React Testing Library.

## Features

- Add todos
- Add todos by pressing Enter
- Toggle todo completion
- Delete todos
- Display todo counter

## Testing

The tests check user-visible behavior:

- Rendering an empty list
- Rendering initial todos
- Adding todos
- Preventing empty todos
- Toggling completed state
- Deleting todos
- Updating the counter
